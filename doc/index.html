NuiForms - Documentation

Welcome to the NuiForms documentation. This library provides minimalist UI components for small applications.

Label

The Label component is used to display text.

Properties

  • text: string - The text to display
  • align: string - Text alignment
  • underline: string - Underline style

TextBox

The TextBox component is used to display a text input field.

Properties

  • text: string - The text value
  • readonly: boolean - Whether the field is read-only
  • hint: string - Placeholder hint text

Button

The Button component is used to display a button.

Properties

  • text: string - The button text
  • onclick: func() - Click handler function
